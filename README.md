# Data 205 Final Project: Bus Stops
### Author: ***Grace Sampson***

### Overview
This project analyzes bus stop infrastructure in the Montgomery County. It's designed to identify areas of lacking infrastructure and suggest specific bus stops to be improved in order to reduce the distance someone has to walk to find a bus stop with a bench. 

### Tools Used
- Python (Google Colab)
- Nearest Neighbors Model

### Project Structure

This repository follows a structured data science workflow:
- data/ -> raw, interim, and processed datasets
- ingestion/ -> data loading and cleaning
- eda/ -> exploratory data analysis
- analysis/ -> hypothesis and Anova testing
- model/ -> Nearest Neighbors model
- reports/ -> final outputs and presentations
